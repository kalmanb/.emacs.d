# Keybindings
Major mode leader is ',' or 'spc m'
Meta key is option on Mac

|key      | description                  |
|---------|------------------------------|
|spc spc  |switch to last buffer         |
|spc g r  |go to .emacs.d/README.md      |
|spc g i  |go to .emacs.d/init.el        |
|spc /    |search
|spc r l  |search resume
